Methodology

Construction & Evaluation Pipeline

We introduce BizGenEval, a systematic benchmark for commercial visual content generation. The benchmark spans five representative document types—slides, charts, webpages, posters, and scientific figures—and evaluates four key capability dimensions: text rendering, layout control, attribute binding, and knowledge-based reasoning, forming 20 diverse evaluation tasks. BizGenEval contains 400 carefully curated prompts and 8,000 human-verified checklist questions to rigorously assess whether generated images satisfy complex visual and semantic constraints.

Figure: Overview of the construction and evaluation pipeline. The system converts real-world references and domain knowledge into structured prompts, and evaluates generated images using rigorous checklists.

Benchmark Design

Summary

Examples

🌐

Webpage

Generate a complete webpage-style visual with clear section hierarchy, aligned UI blocks, and consistent styling across headers, body copy, imagery, and call-to-action regions.

🧪

Scientific Figure

Compose a publication-style scientific figure with multiple coordinated panels, labeled components, and a visually consistent explanation of methods, data flow, or experimental outcomes.

🖥️

Slides

Design a presentation slide with a strong title area, balanced content regions, and tightly aligned chart, text, and annotation elements suitable for executive communication.

📊

Chart

Produce a chart-centric visual that combines accurate axes, legends, and category encoding with readable labels and an overall polished analytic presentation.

🪧

Poster

Assemble a poster-like layout with high information density, strong visual grouping, and readable typography across title, body sections, callouts, and supporting figures.

🧠

Knowledge-based Reasoning

Generate a publication-style figure on Bayes' Theorem and the Base Rate Fallacy — including a probability tree, formal theorem statement, and a visualization demonstrating why most positive results are false positives.

🎨

Attribute Binding

Create a flat 2D infographic with precisely specified color-coded sections, exact icon types per tile, and correctly attributed text for each of seven content pillars.

📐

Layout Control

Create a clean two-panel scientific figure with precisely placed components — backbone, encoder, decoder, prediction heads — connected with specific arrow types.

✍️

Text Rendering

Create an infographic with exact multi-line headings and verbatim paragraph body text placed precisely in six spatial regions.
